# Sample Single Page Application 

powered by [BEAR.Sunday](https://bearsunday.github.io/) and [Vue.js](https://vuejs.org/)

## Setup

``` bash
# install dependencies
composer install

# serve application at 127.0.0.1:8080
composer serve
```

## Requirements

* PHP 7.0+
* composer
* Node.js 6.9+
* yarn
